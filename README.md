A python tool to extract or modified the textures in the PC version of Alias.

Made with the help of Google Gemini.

# Supported Files
- .wld
- _p.cat (the _s.cat store the name and information for each texture)
- .all (with the XPR0 header)

# Supported Versions
- PC
- XBOX

# Instructions
<img width="628" height="38" alt="image" src="https://github.com/user-attachments/assets/5030d9c6-7f8f-433a-add6-939bcaa50fce" />

1. Select "Browse" to select your file.

<img width="600" height="572" alt="image" src="https://github.com/user-attachments/assets/b2375f91-b625-4124-9cb4-c9bd42d7ecb2" />
   
2. Now you can see the textures in the Left Windows, click in one to load it in the Preview Window.

<img width="350" height="82" alt="image" src="https://github.com/user-attachments/assets/0252e82a-12a8-4021-8534-d8531d0a0c50" />

3. Right Click in any texture in the list to open a sub-menu, here you can change the texture, cancel the changes if you made an error or extract the selected texture.
   
4. If you want to change the texture you have to make sure it is with the same compression as the target texture (DXT, Mipmaps, Resolution) or use a texture with a smaller file size. 
   
5. Select <strong>Save File</strong> to save your modified file.
